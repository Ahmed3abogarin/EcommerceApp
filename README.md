# 📱 Swipe to Purchase Button (Jetpack Compose)

This project demonstrates a modern **“Swipe to Purchase”** interaction built entirely using **Jetpack Compose**.  
The goal is to create a smooth, intuitive gesture-based UI with polished animations.

---

## ✨ Features

- 👉 Horizontal drag gesture using `draggable`
- 🎯 Threshold-based swipe detection
- 🎨 Animated background color transition on success
- 🔤 Text fades out smoothly during interaction
- 🔄 Paddle returns smoothly with animation (no sudden jumps)
- 🔁 Fully resettable state for repeated use
- 🎬 Animated icon transition (arrow → check)

---

## 🧠 Focus

This project emphasizes **micro-interactions** and user experience:

- Smooth and natural swipe behavior  
- No abrupt UI changes (everything is animated)  
- Clear visual feedback for success  
- Clean and reusable composable structure  

---

## 🛠️ Tech Stack

- **Kotlin**
- **Jetpack Compose**
- Compose Animations (`animate*AsState`, `AnimatedContent`)
- Gesture handling (`draggable`)

---

## 🚀 Future Improvements

- Add haptic feedback on success  
- Support loading state (e.g., API call before confirming purchase)  
- Convert into a fully reusable composable with parameters  
- Improve accessibility support
